# To-Do List App

A colorful, priority-based To-Do List application built with HTML, CSS, and JavaScript. Tasks are saved in **LocalStorage**, so they persist even after refreshing the page.

## Features
- Add, delete, and mark tasks as completed.
- Assign priority: **Low (Green), Medium (Orange), High (Red)**.
- Tasks saved in LocalStorage.
- Responsive and modern UI with icons.
- Keyboard support: press Enter to add a task.

## Tech Stack
- HTML
- CSS
- JavaScript
- Font Awesome for icons
